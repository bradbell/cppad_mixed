Example Use of Ipopt

Problem
@[@ \begin{array}{ll} \R{minimize} & \beta (x_0 - 2)^2 + \beta (x_1 - 4)^2 \; \R{w.r.t} \; x \in \B{R}^2 \\ \R{subject\; to} & 0 = x_0 + x_1 - 2 \\ & 0 \leq x_0 \leq 2 \\ & 0 \leq x_1 \leq 3 \end{array} @]@
Lagrangian
For @(@ x \in \B{R}^2 @)@, @(@ \lambda \in \B{R} @)@, @(@ L \in \B{R}_+^2 @)@, @(@ U \in \B{R}_+^2 @)@, the Lagrangian is @[@ \beta ( x_0 - 2 )^2 + \beta ( x_1 - 3 )^2 + \lambda_0 ( x_0 + x_1 - 2 ) + U_0 ( x_0 - 2 ) - L_0 x_0 + U_1 ( x_0 - 3 ) - L_1 x_1 @]@

Stationary Conditions
@[@ \begin{array}{rcl} 0 & = & 2 \beta ( x_0 - 2 ) + \lambda_0 - L_0 + U_0 \\ 0 & = & 2 \beta ( x_1 - 3 ) + \lambda_0 - L_1 + U_1 \\ 0 & = & x_0 + x_1 - 2 \end{array} @]@
Solution
If we assume the lower and upper constraints are no active at the solution we get @(@ L = 0 @)@, @(@ U = 0 @)@, @(@ \lambda_0 = 2 \beta (2 - x_0 ) @)@, @(@ x_1 = 2 - x_0 @)@, @[@ \begin{array}{rcl} 0 & = & 2 \beta ( 2 - x_0 - 3 ) + 2 \beta ( 2 - x_0 ) \\ 0 & = & 2 - x_0 - 3 + 2 - x_0 \\ x_0 & = & 0.5 \end{array} @]@ The corresponding solution is @(@ x_0 = 0.5 @)@, @(@ x_1 = 1.5 @)@, and @(@ \lambda_0 = 3.0 @)@. Note that the upper and lower constraints are not active, so this is the solution (with @(@ L = 0 @)@, @(@ U = 0 @)@).
