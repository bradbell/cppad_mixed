Syntax

Private
This is a private member function of the class sparse_ad_cholesky. In addition, this class, and all of its members, are implementation details and not part of the CppAD::mixed user API.

jac_sparsity
This argument has one of the following prototypes
     CppAD::local::sparse_list& jac_sparsity
     CppAD::local::sparse_pack& jac_sparsity
 Its input value does not matter. Upon return it contains the jacobian sparsity pattern for the mapping from the lower triangle of the symmetric matrix Alow to the Cholesky factor L .

n_set
The number of sets jac_sparsity.n_set() is equal to the number of elements in the Cholesky factor @(@ L @)@.

end
The end for each set jac_sparsity.end() is equal to the number of elements in the lower triangle of the symmetric matrix @(@ A @)@. The elements of the set are greater than or equal zero and less than the end value.

elements
If j is an element of set i , then the i-th element of @(@ L @)@ depends on the j-th element of the lower triangle of @(@ A @)@ (using column major ordering).
