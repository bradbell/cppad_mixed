Assumptions

init_laplace_obj_hes_done_
The input value of this member variable must be false. Upon return it is true.

mixed_object

     const CppAD::vector<double>& fixed_vec

This argument has prototype
     const CppAD::vector<double>& random_vec
 It specifies the initial value for the random effects optimization.

laplace_obj_hes_
The input value of the member variable
     CppAD::mixed::sparse_hes_info laplace_obj_hes_
 does not matter. Upon return it contains the sparse_hes_info for the lower triangle of the Hessian @[@ r_{\theta,\theta} ( \theta ) = H_{\beta,\beta} [ \beta, \theta , \hat{u} ( \theta) ] @]@ see H(beta, theta, u) . Note that the matrix is symmetric and hence can be recovered from its lower triangle.

laplace_obj_fun_
This ADFun object should be used in the sparse Hessian Call .
