Ipopt NLP Class Used to Optimize Fixed Effects

Private
This class is an implementation detail and not part of the CppAD::mixed user API.

get_error_message()
This function returns a std::string value that corresponds to the most recent error message. If it is non-empty, there is no error.

clear_error_message()
This function sets the current error message to the empty string.

nlp_lower_bound_inf()
This member function returns the double value used for minus infinity as a lower bound.

nlp_upper_bound_inf()
This member function returns the double value used for plus infinity as an upper bound.

solution()
This member function returns a fixed_solution object containing the optimal solution information.
