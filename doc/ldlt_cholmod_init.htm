void ldlt_cholmod::init(const CppAD::mixed::sparse_rc& H_rc)

Private
The ldlt_cholmod class is an implementation detail and not part of the CppAD::mixed user API.

ldlt_obj
This object has prototype
     CppAD::mixed::ldlt_cholmod ldlt_obj
H_rc
This argument is a sparsity pattern for the square matrices with nrow_ rows that we will compute the LDLT factor of. It is in column major order and lower triangular .

H_rc_
This member variable is set to a copy of H_rc .

Assumptions
All of the cholmod private pointers are null when this routine is called.

sym_matrix_
Upon return, this is set to a packed, real, sorted, lower triangular, sparse matrix with the pattern specified by H_rc and the value nan for each possibly non-zero value.

factor_
Upon return, this is the result of
     factor_ = cholmod_analyze(sym_matrix_, &common_)
rhs_
Upon return, this is set to dense column vector of nrow_ zeros as follows:
     rhs_ = cholmod_zeros(nrow_, 1, CHOLMOD_REAL, &common_)
rhs_set_
Upon return, this is a sparse column vector with nrow_ rows. It is packed, sorted, not symmetric, and just a sparsity pattern. (Note that sorted refers the column order and does not matter for vectors.) There are nrow_ possibly non-zero elements, but which elements are non-zero is not specified (expected to be set for each ldlt_cholmod_solve_H usage).

Order of Operations
This ldlt_obj function must be called once, after the constructor and before any other member functions.

Example
The file ldlt_cholmod.cpp contains an example and test that uses this function.
