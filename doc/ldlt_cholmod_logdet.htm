double ldlt_cholmod::logdet(size_t& negative) const

Private
The ldlt_cholmod class is an implementation detail and not part of the CppAD::mixed user API.

ldlt_obj
This object has prototype
     const CppAD::mixed::ldlt_cholmod ldlt_obj
 In addition, it must have a previous call to ldlt_cholmod_update .

negative
The input value of this argument does no matter. Upon return, it is the number of elements of D that are less than zero.

logdet
This return value is the log of the absolute value of the determinant corresponding to the previous call to ldlt_cholmod_update . If the matrix is singular, logdet is minus infinity.
