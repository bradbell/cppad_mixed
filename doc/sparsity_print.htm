Print a CppAD Internal Sparsity Pattern

Syntax
sparsity_print(label, pattern)

Private
This routine is an implementation detail and not part of the CppAD::mixed user API.

label
This argument has prototype
     const std::string& label
 It is a label printed before the matrix, If it is empty, no label is printed.

pattern
Is the sparsity pattern which must have one of the following prototypes:
          CppAD::local::sparse_pack& pattern
          CppAD::local::sparse_list& pattern
 Note these are effectively const, but are not declared so that the corresponding iterator can be used. (The iterators for these types do not support const.)
