Private
This class is an implementation detail and not part of the CppAD::mixed user API.

Purpose
This class has utilities that work with a cholmod LDLT factor, cholmod is part of the suitesparse package.

Factorization
The factorization is @[@ L D L^\R{T} = P H P^{T} @]@ where

H
is the matrix corresponding the current update .

L
is a lower triangular matrix with ones on the diagonal,

D
is a diagonal matrix.

P
is a permutation matrix.

Example
The file ldlt_cholmod.cpp contains an example and test using the operations in this class.

Preprocessor Symbols
The following extra CHOLMOD_* symbols are defined 
    # define CHOLMOD_TRUE                  1
    # define CHOLMOD_FALSE                 0
    # define CHOLMOD_STYPE_NOT_SYMMETRIC   0
    # define CHOLMOD_STYPE_LOWER_TRIANGLE -1
