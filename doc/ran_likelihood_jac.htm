User Define Jacobian With Respect to Random Effects

Syntax
vec = mixed_object.ran_likelihood_jac(fixed_vec, random_vec)

See Also
ran_likelihood_hes

Purpose
In some cases the user can define a more efficient version of the Jacobian @(@ f_u ( \theta , u ) @)@ than can be obtained by applying AD to the ran_likelihood function. In this case one may define this function directly. For example, it may be possible for the user to defined ran_likelihood_jac using CppAD checkpointing. If one puts checkpointing in ran_likelihood , it would be present in ran_like_a1fun_ but it would be lost when recording derivatives calculated by ran_like_a1fun_.

Public
This cppad_mixed member function is public .

mixed_object
We use mixed_object to denote an object of a class that is derived from the cppad_mixed base class.

Virtual Function
The following virtual function may be implemented in the derived class:
     virtual CppAD::vector<a1_double> ran_likelihood_jac(
          const CppAD::vector<a1_double>& fixed_vec   ,
          const CppAD::vector<a1_double>& random_vec
     )
fixed_vec
This argument has prototype
     const CppAD::vector<Float>& fixed_vec
 It contains a value for the fixed effects vector @(@ \theta @)@.

random_vec
This argument has prototype
     const CppAD::vector<Float>& random_vec
 It contains a value for the random effects vector @(@ u @)@.

vec
The return value has prototype
     CppAD::vector<a1_double>& vec
 Upon return its size is the same as random_vec and it contains the values in the Jacobian. To be specific, vec[j] is the partial of the random likelihood @(@ f( \theta , u ) @)@ with respect to @(@ u_j @)@.

Default
The base class definition (default version) of ran_likelihood_jac returns a size zero vector for vec (the empty vector). In this case, the Jacobian @(@ f_u ( \theta , u ) @)@ is computed by applying AD to the ran_likelihood function.

Derivative Test
If NDEBUG is not defined, the routine ran_like_jac checks that the derivatives agree with the AD derivatives.

Example
The file ran_likelihood_jac.cpp contains an example of defining this virtual function.
