This class is an implementation detail and not part of the CppAD::mixed user API.

Factorization
The factorization is @[@ L D L^\R{T} = P H P^{T} @]@

H
is the matrix corresponding the current update .

L
is a lower triangular matrix with ones on the diagonal,

D
is a diagonal matrix.

P
is a permutation matrix.

Double
This is the type of the elements in the matrices and is either double29628 or a1_double29628.

Example
The file ldlt_eigen.cpp contains an example and test using the operations in this class.
