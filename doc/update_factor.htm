Update the Factorization of Hessian w.r.t. Random Effects

Syntax
mixed_object.update_factor(fixed_vec, random_vec)

Private
This cppad_mixed member function is private .

Purpose
This routine updates the Cholesky factorization of the Hessian @(@ f_{u,u} ( \theta , u )^{-1} @)@ so that it corresponds to the current value of @(@ \theta @)@ and @(@ u @)@.

mixed_object
We use mixed_object to denote an object of a class that is derived from the cppad_mixed base class.

fixed_vec
This argument has prototype
     const CppAD::vector<double>& fixed_vec
 and is the value of fixed effects @(@ \theta @)@.

random_vec
This argument has prototype
     const CppAD::vector<double>& random_vec
 and is the value of fixed effects @(@ u @)@. If this factorization is used for computing derivatives of the Laplace objective , this should be the optimal random effects .

ran_hes_fun_
The ran_hes_fun_ member variable will hold the first order Taylor coefficient corresponding to the specified fixed and random effects; i.e.,
     ran_hes_fun_.Forward(0, both)
 has been called where both is a packed version of the fixed and random effects.

ldlt_ran_hes_
On input, the member variable
     CPPAD_MIXED_LDLT_CLASS ldlt_ran_hes_
 has been initialized using the sparsity pattern for the Hessian. Upon return, ldlt_ran_hes_ contains the updated factorization corresponding to the specified values for the fixed and random effects.

Example
The file update_factor.cpp contains an example and test of this procedure. It returns true, if the test passes, and false otherwise.
