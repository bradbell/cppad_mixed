Compute Log Determinant for Current LDLT Factor

Syntax
logdet = ldlt_obj.logdet(negative)

Private
The ldlt_eigen class is an implementation detail and not part of the CppAD::mixed user API.

ldlt_obj
This object has prototype
     const CppAD::mixed::ldlt_eigen ldlt_obj
 In addition, it must have a previous call to ldlt_eigen_update .

negative
This argument has prototype
     size_t& negative
 Its input value does no matter, upon return it is the number of elements of D that are less than zero.

logdet
This return value has prototype
     double logdet
 Is the log of the absolute value of the determinant corresponding to the previous call to ldlt_cholmod_update . If the matrix is singular, logdet is minus infinity.
