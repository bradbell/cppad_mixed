Compute New Random Effects and Update Factor

Syntax
new_random(fixed_vec)

ipopt_fixed
This is a private member function of the ipopt_fixed class.

n_random_
Is assumed that this member variable is greater than zero.

random_ipopt_options_
This member variable contains the value of the random_ipopt_options in the ipopt_fixed constructor.

random_lower_
This member variable contains the value of the random_lower in the ipopt_fixed constructor.

random_upper_
This member variable contains the value of the random_upper in the ipopt_fixed constructor.

random_in_
This member variable contains the value of the random_in in the ipopt_fixed constructor.

fixed_vec
This argument has prototype
     const d_vector& fixed_vec
 it is the value of the fixed effects that we are computing the random effects and updated factor for.

random_cur_
This member variable contain is set the optimal random effects corresponding to fixed_vec .

mixed_object_
The factor in this member variables is updated using the call
     mixed_object_.update_factor(fixed_vec, random_cur_)
 see update_factor for side effects.
