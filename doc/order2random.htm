Second Order Representation of Random Effects, W(beta, theta, u)

Syntax
W = CppAD::mixed::order2random(
     n_fixed,
     n_random,
     a1fun,
     jac_a1fun,
     ran_hes_rc,
     beta_theta_u
)

Prototype 
    
    a1_vector order2random(
     cppad_mixed&             mixed_object    ,
     size_t                   n_fixed         ,
     size_t                   n_random        ,
     CppAD::ADFun<a1_double>& a1fun           ,
     CppAD::ADFun<a1_double>& jac_a1fun       ,
     const sparse_rc&         ran_hes_rc      ,
     const a1_vector&         beta_theta_u    )

Private
This cppad_mixed member function is private .

mixed_object
is the cppad_mixed object for this problem. This object is only used to call
     mixed_object.ran_likelihood_hes
 and see if the user has defined the hessian of the random likelihood.

n_fixed
number of fixed effects.

n_random
number of random effects

a1fun
This is the random likelihood f(theta, u) .

jac_a1fun
This is the Jacobian of the random likelihood with respect to the random effects @(@ f_u ( \theta, u ) @)@.

ran_hes_rc
This is the sparsity pattern for the lower triangle of the Hessian w.r.t the random effect. The indices in ran_hes_rc are with respect to both the fixed and random effects and hence are greater than or equal n_fixed .

beta_theta_u
This vector has size 2*n_fixed + n_random and specifies the value of (beta, theta, u) at which we are evaluating the second order approximation for the optimal random effects W(beta, theta, u) .

W
The return value is W(beta, theta, u) .
