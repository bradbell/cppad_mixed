Prototype 

a1_vector order2random(
     size_t                              n_fixed         ,
     size_t                              n_random        ,
     CppAD::ADFun<a1_double, double>&    jac_a1fun       ,
     const ldlt_eigen<a1_double>&        a1_ldlt_ran_hes ,
     const a1_vector&                    beta            ,
     const a1_vector&                    theta_u         )

Private
This cppad_mixed member function is private .

n_fixed
number of fixed effects.

n_random
number of random effects

jac_a1fun
This is the Jacobian of the random likelihood with respect to the random effects @(@ f_u ( \theta, u ) @)@. The domain indices (range indices) are with respect to the random effects (fixed and random effects); i.e.,
     jac_a1fun.Domain() == n_fixed + n_random
     jac_a1fun.Range() == n_random
a1_ldlt_ran_hes
This is the LDLT factorization for the for the Hessian w.r.t the random effect; i.e., f_uu ( theta , u ). Note that this object is const and the previous call to update corresponded to (theta, u).

beta
This vector has size n_fixed and is the CppAD independent variable vector at which we are evaluating the second order approximation for the optimal random effects W(beta, theta, u) .

theta_u
This vector has size n_fixed + n_random and is the CppAD independent dynamic parameter vector at which we are evaluating the second order approximation for the optimal random effects W(beta, theta, u) .

W
The return value is W(beta, theta, u) .
