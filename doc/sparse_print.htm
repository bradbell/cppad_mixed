Syntax
sparse_print(label, mat)

Private
This routine is an implementation detail and not part of the CppAD::mixed user API.

Scalar
It is the type for a scalar in the matrix. If s has type Scalar ,
     std::cout << s
 must print the value of s on standard output.

label
This argument has prototype
     const std::string& label
 Is a label printed before the matrix, If it is empty, no label is printed.

mat
Is the sparse matrix which must have one of the following prototypes:
          const Eigen::SparseMatrix<Scalar, Eigen::ColMajor>& mat
          const Eigen::SparseMatrix<Scalar, Eigen::RowMajor>& mat
