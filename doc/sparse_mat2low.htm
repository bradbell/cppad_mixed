Extract the Lower Triangular From an Eigen Symmetric Matrix

Syntax
lower = CppAD::mixed::sparse_mat2low(matrix)

Private
This routine is an implementation detail and not part of the CppAD::mixed user API.

matrix
The argument has prototype
     const Eigen::SparseMatrix<Scalar, Options, Index>& matrix
 and has the same number of rows as columns; i.e.
     matrix.rows() == matrix.cols()
Scalar
The scalar type fro this sparse matrix are arbitrary; i.e, has no restrictions.

Options
The options for this sparse matrix are arbitrary; i.e., has no restrictions.

Index
The index type fro this sparse matrix are arbitrary; i.e., has no restrictions.

lower
The return value has prototype
     Eigen::SparseMatrix<Scalar, Options, Index> lower
 and is the lower triangle of matrix ; i.e., it has the same lower triangle as matrix and it has no entries above the diagonal.

Example
The file sparse_mat2low.cpp is an example and test of sparse_mat2low.
