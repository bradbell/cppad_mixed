mixed_object

     CPPAD_MIXED_LDLT_CLASS ldlt_ran_hes_

 Upon return, the function

 has been called with hes_rc equal to the sparsity pattern for the Hessian of the random likelihood with respect to the random effects; see ldlt_eigen_init . This sparsity information is relative to just the random effects; i.e., the row and column indices are relative to the vector @(@ u ) @)@. For each row and column indices in hes_rc , the corresponding row and column index in ran_hes_rcv_ is n_fixed_ greater.
