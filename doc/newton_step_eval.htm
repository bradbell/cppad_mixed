Using the Newton Step Checkpoint Function

Syntax
newton_checkpoint.eval(a1_theta_u_v, a1_logdet_step)
Prototype 
    
    void newton_step::eval(
     const a1_vector& a1_theta_u_v  ,
     a1_vector&        a1_logdet_step )

Private
This class is an implementation detail and not part of the CppAD::mixed user API.

Restriction
The newton_step_initialize routine must be called before this routine be used.

newton_checkpoint
This is a Newton step object; see newton_checkpoint .

a1_theta_u_v
This is a point at which we are evaluating the Newton step and log determinant. This vector has size n_fixed + 2 * n_random and the order of its elements are @(@ \theta @)@, followed by @(@ u @)@ followed by @(@ v @)@.

a1_logdet_step
This vector has size 1 + n_random . The input value of its elements does not matter. Upon return,
     a1_logdet_step[0]
 is the log of the determinant of @(@ f_{uu} ( \theta , u ) @)@. For j = 1 ,..., n_random_ ,
     a1_logdet_step[j]
 is the @(@ s_{j-1} @)@ where @(@ s @)@ is the Newton step; i.e., @[@ s = f_{uu} ( \theta , u )^{-1} v @]@
